## Stack

Pure HTML, CSS, JavaScript — no framework, no build step.  
Google Fonts: Inter + Playfair Display.  
All files must stay in the same folder for relative navigation links to work.

## Deploy

Drag the entire folder to [Netlify Drop](https://app.netlify.com/drop) for an instant live URL.
